# LutOldMeter — Калькулятор экспозиции для просроченной фотоплёнки

[👉 English version (README.md)](./README.md)

LutOldMeter — это интерактивное веб-приложение на React, Vite и Tailwind CSS для расчета эффективной чувствительности ($ISO_{\text{эфф}}$) просроченной фотоплёнки.

## 📌 Математическая формула

Для оценки падения чувствительности используется следующая формула:

$$ISO_{\text{эфф}} = \frac{ISO_{\text{исх}}}{2^{\left( \frac{\Delta T}{10} \times K \right)}}$$

Где:
* $ISO_{\text{исх}}$ — номинальное ISO плёнки (указанное на коробке).
* $\Delta T$ — срок просрочки в годах (текущий год съемки минус год окончания срока годности).
* $K$ — коэффициент ускорения деградации, зависящий от номинальной чувствительности.
* $2^{(\dots)}$ — математическое выражение экспозиционных стопов (каждый стоп уменьшает ISO в 2 раза).

---

## ⚡ Классификация коэффициента деградации ($K$)

* **До 64 ISO (Низкое):** $K = 0.7 - 0.8$ (теряет меньше 1 стопа за 10 лет).
* **100 – 200 ISO (Среднее):** $K = 1.0$ (классическая потеря 1 стопа за 10 лет).
* **400 ISO (Высокое):** $K = 1.2 - 1.4$ (теряет около 1.3 стопа за 10 лет).
* **800 ISO и выше (Катастрофическое):** $K = 1.8 - 2.0$ (до 2 стопов за 10 лет, высокий уровень вуали).

## 🌟 Основные возможности

1. **Поддержка двух языков:** Мгновенное переключение между английским и русским (английский установлен по умолчанию).
2. **Интерфейс «Liquid Glass»:** Премиальный темный дизайн с эффектом «жидкого стекла», матового стекла (`backdrop-filter`) и анимацией световых бликов.
3. **Учет условий хранения и типа эмульсии:** Настройка коэффициента в зависимости от температуры хранения (Морозилка, Холодильник, Комнатная температура, Жара) и типа фотоматериала (Ч/Б, Цветной негатив, Слайд).
4. **Определение ближайшего стандартного ISO:** Подсказывает ближайшее стандартное значение по шкале 1/3 стопа для выставления на экспонометре камеры.
5. **Интерактивный симулятор кадра:** Сравнение кадра без компенсации (снятого как номинал) и кадра с правильной компенсацией экспозиции.
6. **Пошаговый расчет:** Интерактивный вывод математических шагов в реальном времени.
7. **Таймер проявки и руководство:** Интерактивный таймер процессов со звуковым оповещением для агитации (переворачивания бачка). Расчитывает точное время в зависимости от типа плёнки, структуры зерна/эмульсии (тонкая, средняя, толстая), проявителя (метоловый ПМ, Д-76, Родинал и др.), температуры, разведения и номера использования раствора.

## 🚀 Как запустить локально

### Требования
* [Node.js](https://nodejs.org/) (версии 16+)
* [npm](https://www.npmjs.com/)

### Инструкция

1. Установка зависимостей:
   ```bash
   npm install
   ```

2. Запуск сервера разработки:
   ```bash
   npm run dev
   ```

3. Сборка для продакшена:
   ```bash
   npm run build
   ```
